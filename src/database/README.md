# 数据库技术

数据库是应用系统的核心组件，本模块涵盖关系型数据库和NoSQL数据库。

## 模块概览

| 章节 | 描述 |
|------|------|
| [MySQL](./mysql.md) | 关系型数据库，SQL优化 |
| [Redis](./redis.md) | 内存数据库，缓存技术 |
| [Elasticsearch](./es.md) | 搜索引擎，日志分析 |
