# 设计模式

设计模式是软件开发中经过验证的解决方案，帮助解决常见的设计问题。

## 模式分类

| 类型 | 说明 | 模式 |
|------|------|------|
| 创建型 | 处理对象创建 | 单例、工厂、建造者、原型 |
| 结构型 | 处理类和对象组合 | 适配器、桥接、装饰器、代理等 |
| 行为型 | 处理对象间通信 | 策略、观察者、命令、迭代器等 |
